# LibraryManageSite

프로젝트 팀명 : 도서관 사이트

참여자 : 주민철 - 백엔드 위주로 수행. 프론트 엔드는 기존 예제 활용

기간 : 23.09 ~ 23.12

사용 기술 : 자바, 스프링, SQL, JDBC

프로젝트 목표 : 자바 스프링을 활용한 예제 학습.

주요 특징 : jdk 11 버전 사용. 암호화 된 비밀번호. 비밀번호 수정 가능. 사용자 이메일로 새 비밀번호 전송

주요 기능 : 관리자/사용자 계정 로그인 및 관리. 도서 관리. 대출 및 반납.
